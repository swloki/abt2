# feat: WMS 库存查询 + 入库 + 出库 (Sub-Plan B)

## Summary

实现库存查询列表页、入库管理（列表+新建=2页）、出库管理（列表+新建=2页），共 5 个页面。

---

## Implementation Units

### U9. 库存查询列表页

**原型文件:** `03-stock-list.html`
**侧边栏 key:** `inv-stock`

**Files:**
- Create: `abt-web/src/pages/wms_stock.rs`
- Modify: `abt-web/src/routes/wms.rs`

**Approach:**
- 4 个统计小卡片：
  - 总品种、总库存量、低库存项、**已预留量**（⚠️ 原型第4个是「已预留量」不是「零库存」）
- 搜索框：按产品编码/名称搜索
- 筛选：仓库下拉 + 库区下拉
  - ⚠️ 原型**没有产品分类筛选**，有库区筛选
- **仅显示低库存** 开关（⚠️ 原型是「仅显示低库存」不是「显示零库存」）
- 表格列：产品编码、产品名称、仓库、库区、储位、**批次号**、现有量、**已预留量**、可用量、**单位成本**、安全库存、**入库日期**、**有效期**
  - ⚠️ 原型列比计划多很多：批次号/单位成本/入库日期/有效期
  - ⚠️ 原型没有独立的 status 列，用行高亮代替
- 低库存行高亮

**设计对齐修正：** 统计卡片4、筛选器、表格列名全部以原型为准。

### U10. 入库管理 — 列表页

**原型文件:** `03-stockin-list.html`
**侧边栏 key:** `inv-stockin`

**Files:**
- Modify: `abt-web/src/pages/wms_stock.rs`
- Modify: `abt-web/src/routes/wms.rs`

- 4 个汇总卡片：本月入库单(156) / **入库总金额**(¥892,340) / 待审核(12) / **已完成**(142)
  - ⚠️ 原型第4个卡片是「已完成」（紫色）非「今日入库」
- 状态 Tab：全部(156) / 待审核(12) / 已入库(142) / 已取消(2)
- 筛选栏：搜索单号/物料编码 + **入库类型**(采购入库/生产入库) + **目标仓库** + **日期范围** + 重置
- 表格列(12)：checkbox/入库单号(链接)/入库类型(pill)/来源单号(链接)/目标仓库/物料数量/入库总量/总金额/状态(pill)/操作员/入库时间/操作
  - 入库类型pill：采购入库=蓝色，生产入库=绿色
  - 操作按钮因状态而异：已入库→「详情」，待审核→「审核|取消」，已取消→「详情」
- **导出**按钮 + **新建入库单**按钮
- 蓝色信息栏（流程说明）
- 分页
**设计对齐修正：** 统计卡片、表格列、筛选栏、操作按钮以原型为准。

### U11. 入库管理 — 新建页

**原型文件:** `03-stockin-create.html`

**Files:**
- Modify: `abt-web/src/pages/wms_stock.rs`
- Modify: `abt-web/src/routes/wms.rs`

- 入库类型切换：**采购入库** / **生产入库**（⚠️ 原型只有 2 种类型，没有「其他入库」）
- **来源类型**选择：来料通知(AN) / 采购订单(PO) / 手工录入（⚠️ 原型有3种来源，非仅PO）
  - 来源单号选择后显示来源单信息预览
- 供应商信息区：供应商（自动填充）→ 联系人/联系电话
- 入库信息区：目标仓库、**目标库区**、**目标储位**、**操作员**
  - 上架策略提示条（绿色信息条，显示当前策略如 SAME_MERGE）
- 行项表格：序号/产品编码/产品名称/**规格型号**/批次号/入库数量/**单位成本**/**小计**/**目标储位**（显示「自动分配」）/[删除]
- **入库汇总**区：物料种类/入库总量/入库总金额/上架策略
- **备注** textarea
- 添加行 / 删除行
- 底部：保存草稿 / **确认入库**
**设计对齐修正：** 来源类型有3种（AN/PO/手工）；有额外的送货单号/库区/储位/操作员/上架策略提示/入库汇总/备注等字段。

### U12. 出库管理 — 列表页

**原型文件:** `03-stockout-list.html`
**侧边栏 key:** `inv-stockout`

**Files:**
- 4 个汇总卡片：本月出库单、**出库总金额**、待拣货、已完成
  - ⚠️ 原型第4个卡片是「已完成」不是「已出库」
- 状态 Tab：全部 / **待拣货** / **拣货中** / **已出库** / **已取消**
  - ⚠️ 原型用仓库操作流程状态（待拣货/拣货中/已出库）
- 筛选栏：搜索 + **出库类型**下拉 + **来源仓库**下拉 + 日期范围 + 重置
- 表格列：checkbox/出库单号/出库类型(彩色pill)/来源单号/来源仓库/物料数量/出库总量/总金额/拣货策略/状态(pill)/操作员/出库时间/操作
  - 出库类型颜色：销售出库=红色pill，生产领料=橙色pill
  - 操作按钮因状态而异：已出库/已取消→「详情」，待拣货→「拣货|取消」，拣货中→「确认出库」
- **导出**按钮 + **新建出库单**按钮
- 红色提示栏（流程说明：SOFT/HARD预留、FIFO/FEFO策略）
- 分页
**设计对齐修正：** 出库列表比入库复杂很多，有完整表格、条件操作按钮、导出功能。

### U13. 出库管理 — 新建页

**原型文件:** `03-stockout-create.html`

**Files:**
- Modify: `abt-web/src/pages/wms_stock.rs`
- Modify: `abt-web/src/routes/wms.rs`

- 出库类型切换：**销售出库**(SALES_SHIPMENT/SOFT预留) / **生产领料**(MATERIAL_ISSUE/HARD预留)
  - 类型描述卡片包含预留类型说明
- **来源关联区**：来源类型(发货申请SH/领料单MR/**手工录入**) + 来源单号(选择按钮) + 客户/工单(readonly) + 预留类型(readonly, SOFT/HARD)
  - 来源单信息预览（来源单号+订单号+预留状态）
- **出库信息区**：来源仓库* + **来源库区** + **拣货策略**(FIFO/FEFO/最短路径/整托优先) + 操作员
  - 拣货策略提示条（说明当前策略）
- **出库物料明细表格**(11列)：序号/产品编码/产品名称/规格型号/批次号/出库数量*/**可用库存**(✓ok/⚠warn/✗danger)/**单位成本**/**小计**(自动计算)/**来源储位**/[删除]
  - **可用库存检查**：avail > qty → ✓绿色，接近 → ⚠琥珀，不足 → ✗红色
- **库存预留&可用性检查区**：预留类型/已预留量/出库后释放说明
- **出库汇总**：物料种类/出库总量/**出库总金额**(红色)/拣货策略 — 自动计算
- **备注** textarea
- 添加行 / 删除行
- 底部：保存草稿 / **确认出库**
