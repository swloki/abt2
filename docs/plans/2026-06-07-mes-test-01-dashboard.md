## 2. 统计卡片区域

### 2.1 卡片内容

| ID | 卡片名称 | 图标颜色 | 数据字段 | 原型期望 |

| DASH-14 | 本月完工数量 | 渐变绿色 | completed_qty | 显示具体数字（如 356） |

### 2.2 测试要点

| ID | 测试项 | 预期结果 |
|----|--------|---------|
| DASH-20 | 卡片渲染 | 5 个卡片正确显示，每个包含 SVG 图标 + 数值 + 标签 |

## 4. 最近操作区域

### 4.1 原型设计

原型中有"最近操作"表格，表头：

| 列 | 说明 |
|----|------|
| 时间 | 操作时间 |
| 操作类型 | 状态 pill（报工/完工入库/工单下达/生产报检/委外发料） |
