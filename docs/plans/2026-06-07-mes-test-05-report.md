### 2.4 工资计算（原型）

原型中有 JS 自动计算：
```javascript
function calcWage() {
    const qty = parseFloat(document.getElementById('completedQty').value) || 0;
    const price = parseFloat(unitPriceEl.dataset.price) || 0;
    document.getElementById('wageAmount').textContent = '¥' + (qty * price).toFixed(2);
}
```

> **实现**: 没有 wage 自动计算显示。

### 4.2 原型设计

原型包含：
1. **筛选栏**: 搜索工人/日期范围/班组筛选
2. **工资汇总卡片(4个)**:
   - 本月工资总额 ¥28,645.50
   - 计件工人数 18
   - 总完成数量 14,820
   - 扣减金额(操作失误) ¥682.00
3. **工资计算公式**: (完成数 + 非操作失误不良数) × 计件单价
4. **工人汇总表格**: 工人/完成数/不良品数/有效计件数/总工时/应发工资
5. **展开明细**: 每个工人的工序明细
6. **导出按钮**
