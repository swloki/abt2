| ID | 字段 | 控件 | name | 必填 | 选项/说明 |

| 原型字段 | 实现状态 | 说明 |

| 工序选择器 | ❌ 缺失 | 原型用 select |
| 产品（自动填充） | ❌ 缺失 | 选择工单后自动填充 |
| 检验员选择器 | ❌ 缺失 | 原型用 select |
| 合格数量 | ❌ 缺失 | number |
| 不合格数量 | ❌ 缺失 | number |
| 检验结果 | ❌ 缺失 | select 合格/不合格/让步接收 |
| 备注 | ❌ 缺失 | CreateInspectionReq 有 remark |
