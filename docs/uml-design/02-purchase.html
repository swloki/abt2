Purchase Module — UML Class Diagram

业务流转

Supplier （Master Data） ──→ PurchaseQuotation 采购报价（比价） ──→ PurchaseOrder 采购订单 ──→ PurchaseReconciliation 对账付款 ──→ PaymentRequest 付款申请
↗ PurchaseReturn 采购退货（逆向）    ↗ MiscellaneousRequest 零星请购（非生产物资）

Supplier 供应商 （→ Master Data 模块）

供应商主档案已统一归入 主数据模块（SupplierService），本模块通过 supplier_id FK 引用。创建订单时调用 MasterData.SupplierService.get() 校验供应商状态

PurchaseQuotation 采购报价

功能：登记各供应商商品单价及严格报价有效期，实现阳光比价

entity PurchaseQuotation {
  id: i64 PK
  doc_number: String // PQ-2026-05-xxxxx
  supplier_id: i64 FK→suppliers
  quotation_date: Date
  valid_from: Date
  valid_until: Date // 报价有效期
  status: PurchaseQuotationStatus
  remark: String
  operator_id: i64 FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseQuotationItem {
  id: i64 PK
  quotation_id: i64 FK→purchase_quotations
  product_id: i64 FK→products
  line_no: i32
  unit_price: Decimal(18,6)
  min_order_qty: Decimal(18,6)? // 最小起订量
  lead_time_days: i32? // 交货周期
  currency: String // 币种
  is_preferred: bool // 是否首选供应商
}

enum PurchaseQuotationStatus {
  DRAFT, ACTIVE, EXPIRED, CANCELLED
  }

共享层：DocumentSequence 编号 | 同一物料多供应商报价可横向比价

PurchaseOrder 采购订单

功能：依据物料短缺或安全库存，向供应商正式下达购买指令

entity PurchaseOrder {
  id: i64 PK
  doc_number: String // PO-2026-05-xxxxx
  supplier_id: i64 FK→suppliers
  order_date: Date
  expected_delivery_date: Date?
  status: PurchaseOrderStatus
  total_amount: Decimal(20,4)
  payment_terms: String?
  delivery_address: String?
  remark: String
  operator_id: i64 FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseOrderItem {
  id: i64 PK
  order_id: i64 FK→purchase_orders
  line_no: i32
  product_id: i64 FK→products
  description: String
  quantity: Decimal(18,6)
  unit_price: Decimal(18,6)
  amount: Decimal(20,4)
  received_qty: Decimal(18,6) // 已收货量
  inspected_qty: Decimal(18,6) // 已检验量
  returned_qty: Decimal(18,6) // 已退货量
  quotation_item_id: i64? FK→purchase_quotation_items
  expected_delivery_date: Date?
}

enum PurchaseOrderStatus {
  DRAFT, CONFIRMED, PARTIALLY_RECEIVED
  RECEIVED, CLOSED, CANCELLED
  }

共享层：DocumentSequence 编号 | InventoryReservation 到货后释放安全库存预留 | DocumentLink 关联报价 | CostEntry 记材料成本

SRM 模块 × 共享层交互汇总

DocumentSequence → PQ/PO/PRT/REC/PAY/MISC 编号
DocumentLink → 报价→订单→入库→对账→付款 全链路关联
InventoryReservation → 安全库存预留 → 到货后释放
CostEntry → 入库记材料成本(借:存货/贷:应付) | 退货冲减 | 付款记流出
QMS → 来料入库前 IQC 硬门检查 is_passed(ArrivalNotice) | 不合格触发 MRB
FMS → CashJournal 采购付款 | WriteOff 按入库单核销 | 三单匹配(入库+发票+付款)
