采购模块 (SRM) — 类图设计

6 个核心实体 + 共享层引用 + 对账付款闭环

Purchase Module — UML Class Diagram

业务流转

Supplier 供应商维护 ──→ PurchaseQuotation 采购报价（比价） ──→ PurchaseOrder 采购订单 ──→ PurchaseReconciliation 对账付款 ──→ PaymentRequest 付款申请
↗ PurchaseReturn 采购退货（逆向）    ↗ MiscellaneousRequest 零星请购（非生产物资）

Supplier 供应商

功能：维护供应商主档案，支持准入审核与分级分类，全生命周期管控

entity Supplier {
  id: UUID PK
  code: String // 供应商编码
  name: String
  short_name: String?
  category: SupplierCategory
  status: SupplierStatus
  tax_number: String?
  legal_person: String?
  registered_capital: Decimal?
  business_scope: String?
  qualification_expiry: Date?
  lead_time_days: i32 // 默认交货天数
  payment_terms: String?
  remark: String
  operator_id: UUID FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity SupplierContact {
  id: UUID PK
  supplier_id: UUID FK→suppliers
  name: String
  position: String?
  phone: String?
  email: String?
  is_primary: bool
}

entity SupplierBankAccount {
  id: UUID PK
  supplier_id: UUID FK→suppliers
  bank_name: String
  account_name: String
  account_number: String
  is_default: bool
}

enum SupplierCategory {
  RAW_MATERIAL, PACKAGING, OUTSOURCING
  CONSUMABLE, SERVICE
  }
enum SupplierStatus {
  PROSPECTIVE, QUALIFIED, PROBATION
  DISQUALIFIED, BLACKLISTED
  }

共享层：DocumentSequence 不适用（编码走主数据治理）| 未来对接 MDM

PurchaseQuotation 采购报价

功能：登记各供应商商品单价及严格报价有效期，实现阳光比价

entity PurchaseQuotation {
  id: UUID PK
  doc_number: String // PQ-2026-05-xxxxx
  supplier_id: UUID FK→suppliers
  quotation_date: Date
  valid_from: Date
  valid_until: Date // 报价有效期
  status: PurchaseQuotationStatus
  remark: String
  operator_id: UUID FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseQuotationItem {
  id: UUID PK
  quotation_id: UUID FK→purchase_quotations
  product_id: UUID FK→products
  line_no: i32
  unit_price: Decimal(10,6)
  min_order_qty: Decimal(10,6)? // 最小起订量
  lead_time_days: i32? // 交货周期
  currency: String // 币种
  is_preferred: bool // 是否首选供应商
}

enum PurchaseQuotationStatus {
  DRAFT, ACTIVE, EXPIRED, CANCELLED
  }

共享层：DocumentSequence 编号 | 同一物料多供应商报价可横向比价

PurchaseOrder 采购订单

功能：依据物料短缺或安全库存，向供应商正式下达购买指令

entity PurchaseOrder {
  id: UUID PK
  doc_number: String // PO-2026-05-xxxxx
  supplier_id: UUID FK→suppliers
  order_date: Date
  expected_delivery_date: Date?
  status: PurchaseOrderStatus
  total_amount: Decimal(10,6)
  payment_terms: String?
  delivery_address: String?
  remark: String
  operator_id: UUID FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseOrderItem {
  id: UUID PK
  order_id: UUID FK→purchase_orders
  line_no: i32
  product_id: UUID FK→products
  description: String
  quantity: Decimal(10,6)
  unit_price: Decimal(10,6)
  amount: Decimal(10,6)
  received_qty: Decimal(10,6) // 已收货量
  inspected_qty: Decimal(10,6) // 已检验量
  returned_qty: Decimal(10,6) // 已退货量
  quotation_item_id: UUID? FK→purchase_quotation_items
  expected_delivery_date: Date?
}

enum PurchaseOrderStatus {
  DRAFT, CONFIRMED, PARTIALLY_RECEIVED
  RECEIVED, CLOSED, CANCELLED
  }

共享层：DocumentSequence 编号 | InventoryReservation 到货后释放安全库存预留 | DocumentLink 关联报价 | CostEntry 记材料成本

PurchaseReturn 采购退货

功能：不合格品退回供应商，冲减应付和库存

entity PurchaseReturn {
  id: UUID PK
  doc_number: String // PRT-2026-05-xxxxx
  order_id: UUID FK→purchase_orders
  supplier_id: UUID FK→suppliers
  return_date: Date
  status: PurchaseReturnStatus
  return_reason: String
  total_amount: Decimal(10,6)
  remark: String
  operator_id: UUID FK→users
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseReturnItem {
  id: UUID PK
  return_id: UUID FK→purchase_returns
  order_item_id: UUID FK→purchase_order_items
  product_id: UUID FK→products
  returned_qty: Decimal(10,6)
  unit_price: Decimal(10,6)
  amount: Decimal(10,6)
}

enum PurchaseReturnStatus {
  DRAFT, CONFIRMED, SHIPPED, REFUNDED
  CANCELLED
  }

共享层：DocumentLink REFERENCES 原采购单 | CostEntry 冲减应付

PurchaseReconciliation 对账单

月底核对入库明细，确认应付

entity PurchaseReconciliation {
  id: UUID PK
  doc_number: String
  supplier_id: UUID FK→suppliers
  period: String // "2026-05"
  status: ReconciliationStatus
  total_amount: Decimal(10,6)
  confirmed_amount: Decimal(10,6)
  difference: Decimal(10,6)
  remark: String
  operator_id: UUID
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity PurchaseReconItem {
  id: UUID PK
  reconciliation_id: UUID
  order_id: UUID
  order_item_id: UUID
  received_qty: Decimal(10,6)
  unit_price: Decimal(10,6)
  amount: Decimal(10,6)
  confirmed: bool
}

PaymentRequest 付款申请

三单匹配：入库+发票+付款

entity PaymentRequest {
  id: UUID PK
  doc_number: String
  supplier_id: UUID FK→suppliers
  reconciliation_id: UUID? FK→purchase_reconciliations
  payment_date: Date
  amount: Decimal(10,6)
  status: PaymentStatus
  payment_method: PaymentMethod
  bank_account_id: UUID? FK→supplier_bank_accounts
  invoice_number: String?
  invoice_amount: Decimal(10,6)?
  remark: String
  operator_id: UUID
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

enum PaymentStatus {
  DRAFT, APPROVED, PAID
  CANCELLED
  }
enum PaymentMethod {
  BANK_TRANSFER, CASH, NOTE
  }

MiscellaneousRequest 零星请购

非生产物资的敏捷采购通道

entity MiscellaneousRequest {
  id: UUID PK
  doc_number: String
  department_id: UUID FK→departments
  request_date: Date
  status: MiscRequestStatus
  total_amount: Decimal(10,6)
  purpose: String // 用途说明
  remark: String
  operator_id: UUID
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity MiscRequestItem {
  id: UUID PK
  request_id: UUID
  line_no: i32
  item_name: String
  specification: String?
  quantity: Decimal(10,6)
  unit: String
  estimated_price: Decimal(10,6)?
  remark: String?
}

enum MiscRequestStatus {
  DRAFT, APPROVED, PURCHASING
  RECEIVED, CLOSED, CANCELLED
  }

SRM 模块 × 共享层交互汇总

DocumentSequence → PQ/PO/PRT/REC/PAY/MISC 编号
DocumentLink → 报价→订单→入库→对账→付款 全链路关联
InventoryReservation → 安全库存预留 → 到货后释放
CostEntry → 入库记材料成本(借:存货/贷:应付) | 退货冲减 | 付款记流出
