WMS Module — UML Class Diagram

核心业务流

ArrivalNotice 来料通知 → IQC检验 → Putaway 上架入库 → StockLedger 库存账
← Pick/Ship 拣货出库 ← MaterialRequisition 生产领料 ← Backflush 系统倒冲
Transfer 调拨 | FormConversion 形态转换 | CycleCount 盘点 | InventoryLock 锁库

三级库位模型

Warehouse 仓库

entity Warehouse {
  id: i64 PK
  code: String
  name: String
  warehouse_type: WarehouseType
  status: WarehouseStatus
  address: String?
  manager_id: i64? FK→users
  is_virtual: bool // 虚拟仓(委外)
  remark: String
  operator_id: i64

enum WarehouseType {

  }
enum WarehouseStatus {
  ACTIVE, INACTIVE
  }

StockLedger 库存账（实时）

物料×批次×储位的实时库存数量

entity StockLedger {
  id: i64 PK
  product_id: i64 FK→products
  warehouse_id: i64 FK→warehouses
  zone_id: i64 FK→zones
  bin_id: i64 FK→bins
  batch_no: String?
  quantity: Decimal(18,6) // 现有量
  reserved_qty: Decimal(18,6) // 已预留量
  available_qty: Decimal(18,6) // = 现有量 - 已预留量
  unit_cost: Decimal(18,6)? // 移动加权平均成本
  received_date: Date?
  expiry_date: Date?
  updated_at: Timestamp
}

  }

供应商送货后进入待检区，不计入可用库存，质检合格后转采购入库

entity ArrivalNotice {
  id: i64 PK
  doc_number: String // AN-2026-05-xxxxx
  purchase_order_id: i64? FK→purchase_orders
  supplier_id: i64 FK→suppliers
  arrival_date: Date
  status: ArrivalStatus
  warehouse_id: i64 FK→warehouses
  zone_id: i64? FK→zones (待检区)
  delivery_note: String? // 送货单号
  remark: String
  operator_id: i64
  created_at: Timestamp
  updated_at: Timestamp
  deleted_at: Timestamp?
}

entity ArrivalNoticeItem {
  id: i64 PK
  notice_id: i64
  order_item_id: i64?
  product_id: i64
  declared_qty: Decimal(18,6) // 供应商申报量
  received_qty: Decimal(18,6) // 实收量
  accepted_qty: Decimal(18,6) // 检验合格量
  batch_no: String?
}

enum ArrivalStatus {
  DRAFT, RECEIVED, INSPECTING, ACCEPTED
  PARTIALLY_ACCEPTED, REJECTED
  }

InventoryTransactionService ..> MasterData.ProductService : product dimension ArrivalNoticeService ..> MasterData.ProductService : product info BackflushService ..> MasterData.BomService : BOM explosion MaterialRequisitionService ..> MasterData.ProductService : product info

WMS 模块 × 共享层交互汇总

DocumentSequence → AN/MR/BF/CC/TRF/FC/LCK 等所有单据编号
DocumentLink → 来料→采购单 | 领料→工单 | 出库→发货申请 | 倒冲→工单
InventoryReservation → 领料消耗HARD预留 | 发货消耗SOFT预留 | 盘点调整后校准
CostEntry → 入库记材料成本 | 出库记销售成本 | 倒冲差异记损耗成本 | 形态转换成本分摊
QMS → 来料入库前 IQC 硬门 is_passed(ArrivalNotice) | 不合格阻断入库
OM → 委外=虚拟库位 | 发外协=InventoryTransfer(到虚拟仓) | 收回=从虚拟仓调回
FMS → 库存成本为 CostEntry 核心数据源 | CostAccounting 聚合查询基础
