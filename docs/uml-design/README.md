## 设计原则

- **共享层先行**：DocumentSequence / DocumentLink / InventoryReservation / CostEntry 为所有模块共享
- **分层包结构**：共享层独立，各模块通过 DocumentType 枚举解耦
- **业财一体**：从第一天起记录成本，避免事后对账
- **委外统一**：委外供应商 = WMS 虚拟库位，复用已有调拨/入库模型
