
# 修改人工成本方式。

~~~ 当前的人工成本管理比较粗糙，而且每个BOM都要单独设置，有一个人工成本变了，要去所有的BOM修改人工成本。~~~

1.  首先有一个工序列表管理，
    - 这个列表里可以给添加各种工序名称，工序名称唯一。
    - 这里的工序可以设置价格。
    - 实现这个列表的crud。
2. 工序组管理。比如说电源工序集，模组工序集
    - 这个主要功能，就是选中工序列表中的工序，组合成真实生的工序。
3. BOM设置人工成本
    - BOM设置人工成本的时候，只要选中工序管理的工序集，选中后，就会列出选中后的工序集，然后设置数量。如果数量为0 ，要填写相应的备注告知
