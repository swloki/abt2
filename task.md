
# 完善人工成本设计。
现在我的人工成本这样设计，有一个bug，就是可能当上传的时候，有一道工序漏掉的时候，而没有办法反馈，我们现在增加新的设计功能来防止这点，

- 还是有一个工序列表，这个列表维护所有工序名称，并且每个工序有一个编码。
- 还有一个工序组，这个工序也是像之前组合工序，保证每个bom人工所需要工序。
- 每一个BOM都需要设置一个工序组。
- 上传工序的时候，首页根据产品产品编码，找出BOM，再找出相应的工序组，根据工序组里的工序编码，匹配上传的工序编码，然后设置相应的单价，数量，如果数量为0的时候，代表不需要这道工序 ，需要设置说明，如果匹配到足够的工序 ，就可以证明工序缺失。
